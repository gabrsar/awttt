How to use AWTTT - API Specification
--------------------------------------------------------------------------------

- UTILS ------------------------------------------------------------------------
GET     / - Show an explanation of what is AWTTT
GET     /help - Show this help message
GET     /healthCheck - return an "I'm alive!" if server is healthy and running

- GAMES (NOT IMPLEMENTED YET) --------------------------------------------------
GET     /games - List all games
GET     /games/:id - Get an specific game
POST    /games - Create a game and return it
